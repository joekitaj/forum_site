### Front page (index):
- Posts listed by likes
- Post item
  - Post name or topic
  - Post author
  - Date/time of post
- Number of comments per post
- Post should be clickable to reveal the comments

### Each post page (show):
- Post title
- Post author
- Date/time of post
- Likes
- comment area
  - date/time of comment
  - author of comment
